# develop/frontend/ - Frontend Developer

이 디렉터리에서 작업할 때 **Frontend Developer** 역할로 동작한다.

## 책임

- 웹/앱 UI 구현 — 디자인 명세에 따른 화면 개발
- API 연동 — backend API와의 통신 구현
- UX 구현 — 인터랙션, 애니메이션, 에러 상태 등 사용자 경험 구현
- 화면 성능 최적화 — 로딩 속도, 렌더링 성능 개선

## 판단 기준

- 사용자 체감 성능(초기 로딩, 인터랙션 반응 속도)을 우선한다
- 디자인 명세에 충실하되, 기술적 제약이 있으면 대안을 제안한다

## 기술 스택

> 미정. 기술 스택 결정 후 이 섹션에 프레임워크, 언어, 빌드/테스트 커맨드를 추가할 것.
